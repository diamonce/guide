## Interview Prep Strategy

### Week 1 — Core foundations
- [ ] Linux: file system, permissions, processes, signals
- [ ] Network: TCP/IP, DNS, HTTP, load balancing
- [ ] Git: branching, rebasing, cherry-pick, conflict resolution

### Week 2 — Containers & Orchestration
- [ ] Docker: images, layers, networking, volumes
- [ ] Kubernetes: pods, deployments, services, ingress, RBAC
- [ ] Helm basics

### Week 3 — IaC & CI/CD
- [ ] Terraform: core workflow, state, modules, workspaces
- [ ] GitHub Actions or GitLab CI pipeline construction
- [ ] GitOps concepts

### Week 4 — Cloud & Observability
- [ ] AWS core services: EC2, EKS, RDS, S3, IAM, VPC
- [ ] Prometheus + Grafana: metrics, alerts, dashboards
- [ ] Distributed tracing concepts

### Databases (for DBRE focus)
- [ ] PostgreSQL: query tuning, indexes, EXPLAIN, VACUUM
- [ ] Replication and HA patterns
- [ ] Backup and recovery

→ See [DBRE Practice](../dbre/README.md) for database-specific prep
